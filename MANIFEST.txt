nazar-studio — native ANSI avatar renderer.

AVATARS (768x768 3x3 sheets; 9 frames; frame0 = still rest):
  assets/avatars/nazar.png       cosmic eye; blink/think frames
  assets/avatars/nazar-expr.png  expression sheet
  assets/avatars/soul.png        Seeker soul-of-light face; typing pulse
  assets/avatars/tools/eye-*.png tool/domain icons; glow pulse while running

RENDERER:
  lib/ui/pixel-avatar.ts    samples the PNG masters directly
  lib/ui/sextant.ts         pure TypeScript mosaic renderer
  lib/ui/graphics-state.ts  low | medium | high quality config

QUALITY:
  low     half-block
  medium  sextant (default)
  high    octant

VALIDATE:
  npm run typecheck && npm test
  npm run review:avatars
