# `termux-setup`

Agent Skill contributed by `@nazar/core` for installing, configuring, auditing, and repairing Nazar on Android/Termux hosts.

Use it when setting up S25 Ultra, Pixel 6a, or other Termux peers with Syncthing-backed Nazar vault/session sync and host-local `AGENTS.md` / `current_host.md` context.
